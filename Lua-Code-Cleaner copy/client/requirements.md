## Packages
prismjs | Syntax highlighting for code blocks
react-simple-code-editor | Lightweight code editor component
date-fns | formatting timestamps for history
framer-motion | Smooth UI transitions and animations
clsx | Utility for conditional classes
tailwind-merge | Utility for merging tailwind classes

## Notes
Integration with PrismJS requires importing a theme CSS in index.css
The app uses a split-pane layout for code editing
Code cleaning happens via POST /api/clean
